# 🛸 БПЛА Аналітика: Навігаційне Ядро

Цей проєкт реалізує інтелектуальну систему обробки телеметрії безпілотних літальних апаратів. Основний фокус зроблено на точності фізичних метрик шляхом поєднання даних GPS та інерціальних датчиків (IMU).

---

## 🧠 Технічна реалізація (Analytics Core)

На відміну від спрощених методів розрахунку, наше ядро використовує принципи побудови **Інерціальних Навігаційних Систем (INS)**:

### 1. 3D-Орієнтація та Кватерніони (AHRS)
Для відстеження положення дрона в просторі ми використовуємо **кватерніони** (`scipy.spatial.transform.Rotation`). 
* **Чому це важливо:** Це дозволяє уникнути ефекту "Gimbal Lock" (заклинювання осей), який виникає при використанні кутів Ейлера під час різких нахилів БПЛА.
* **Initial Alignment:** Система автоматично визначає вектор гравітації на старті та вирівнює віртуальний горизонт.

### 2. Компенсація гравітації (Gravity Compensation)
Алгоритм динамічно перераховує вектор прискорення з локальної системи координат дрона у глобальну систему координат Землі. Це дозволяє:
* Точно віднімати земне тяжіння ($9.81 \text{ м/с}^2$).
* Отримувати **чисте лінійне прискорення** (тягу моторів) без впливу нахилів корпусу.

### 3. Чисельне інтегрування (Trapezoidal Method)
Розрахунок миттєвої швидкості базується на **методі трапецій**:
$$v_i = v_{i-1} + \frac{a_i + a_{i-1}}{2} \cdot \Delta t$$
Це забезпечує вищу точність опису динамічних маневрів у порівнянні зі стандартним прямокутним інтегруванням.

### 4. Sensor Fusion (Complementary Filter)
Ми реалізували алгоритм комплексування даних для подолання недоліків окремих сенсорів:
* **GPS** забезпечує стабільність на довгих дистанціях (не має дрейфу).
* **IMU** забезпечує високу частоту та реакцію на миттєві ривки.
* **Коефіцієнт Fusion ($\alpha = 0.02$):** дозволяє отримати плавну "живу" швидкість, нівелюючи накопичення помилки інтегрування (Drift).

## 📊 Ключові метрики польоту

| Метрика | Опис |
| :--- | :--- |
| **Max Horizontal Speed** | Максимальна швидкість у горизонтальній площині (Fused GPS+IMU) |
| **Max Vertical Speed** | Найвища швидкість набору або скидання висоти |
| **Max Acceleration** | Пікове лінійне перевантаження (G-force), очищене від гравітації |
| **Max Altitude Gain** | Сумарний набір висоти за всю місію (кумулятивна робота підйому) |
| **Total Distance** | Реальний пройдений шлях з урахуванням кривизни Землі (Haversine) |
