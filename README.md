# Raccoonteam
## Метрики польоту (`calculate_metrics`)

Функція `calculate_metrics` обчислює ключові метрики польоту на основі даних з лог-файлу (переданих у вигляді `DataFrame`).  
Всі обчислення проводяться автоматично:  

- **Швидкості** — через інтегрування прискорень (`ax, ay, az`) методом трапецієвидного інтегрування  
- **Дистанція** — через координати (`lat, lon`) за формулою Haversine  
- **Висота** — обчислення максимального набору висоти з колонки `height`  

### Повертає словник з наступними ключами:

| Ключ                     | Опис                                         | Одиниці      |
|---------------------------|---------------------------------------------|-------------|
| `max_horizontal_speed`    | Максимальна горизонтальна швидкість         | м/с         |
| `max_vertical_speed`      | Максимальна вертикальна швидкість           | м/с         |
| `max_acceleration`        | Максимальне прискорення                     | м/с²        |
| `max_altitude_gain`       | Максимальний набір висоти                   | м           |
| `total_distance`          | Загальна пройдена дистанція                 | м           |
| `flight_duration`         | Тривалість польоту                          | с           |

Ця інформація дозволяє Streamlit-розробнику візуалізувати ключові показники польоту незалежно від того, як саме формуються дані лог-файлу.
