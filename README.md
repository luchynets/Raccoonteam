# 🌐 БПЛА Візуалізація: Панель 3D-Аналітики

Цей модуль реалізує інтерактивний інструмент для просторової реконструкції польотних місій. Система перетворює масиви телеметрії у динамічну 3D-сцену, дозволяючи візуально оцінити якість керування та фізику руху дрона.

---

## 🛠 Технічна реалізація (Visualization Engine)

На відміну від статичних 2D-карт, наш інструмент будує повну просторову модель польоту, використовуючи наступні методи:

### 1. Геодезична трансформація (WGS-84 to ENU)
Для коректного відображення координат у метрах ми відмовилися від прямого використання широти/довготи на графіку. Використовується бібліотека `pymap3d`:
* **Система координат:** ENU (East-North-Up).
* **Локальний Datum:** Точка зльоту ($lat_0, lon_0, alt_0$) стає центром координат $[0, 0, 0]$.
* **Результат:** Всі маневри відображаються у реальних фізичних метрах, що дозволяє точно оцінити радіуси поворотів та вертикальні переміщення.

### 2. Градієнтне колорування траєкторії
Траєкторія візуалізується не просто лінією, а як колірна карта швидкостей:
* **Спектр Jet:** Колір треку динамічно змінюється залежно від параметра `fused_speed`.
* **Візуальний аудит:** Сині ділянки відповідають низькій швидкості (зависання/маневрування), червоні — максимальному розгону на прямих.

### 3. Просторова орієнтація та Проекція ("Shadow")
Для вирішення проблеми сприйняття глибини в 3D додано:
* **Ground Projection:** "Тінь" маршруту на площині $Z_{min}$, яка допомагає зрозуміти, над якою саме точкою земної поверхні знаходився БПЛА.
* **Vertical Drop Lines:** Вертикальні зв'язки (маркери) між реальним положенням та проекцією на землю, що створюють ефект об'єму.

## 📊 Інтерактивний HUD (Heads-Up Display)

На графіку реалізовано динамічну панель звіту, яка виводить обчислені метрики безпосередньо у вікно візуалізації:

* **Час у польоті:** Загальна тривалість місії.
* **Дистанція:** Сумарний пройдений шлях.
* **Набір висоти:** Максимальна амплітуда підйому.
* **V-Max (Гор/Верт):** Пікові швидкості в обох площинах.
* **G-Force:** Максимальне прискорення системи.

## 🧰 Використані інструменти

* **Plotly (graph_objects)** — побудова високопродуктивної інтерактивної 3D-графіки з підтримкою анотацій.
* **PyMap3D** — виконання точних геодезичних перетворень координат.
* **Pandas & NumPy** — попередня обробка даних, очищення від нульових GPS-координат та векторизація обчислень.
* **Sensor Fusion Output** — візуалізація результатів роботи інтелектуального навігаційного ядра.
